Exercise, WEB1 session 1

  1. Make a new folder for this session
  2. Download "Star Pizza.txt" and mark up the text.
  3. Add more items to the menu (e.g. pasta, beverages)
  4. Organize menu items into groups with headlines (Pizzas, Pasta, and Beverages).
  5. Use the pizza images found in the images-subfolder to create an appetizing menu for Star Pizza.
  6. Use Paint or another picture editing tool to resize (make a small version) the images of pasta dishes and beverages and add these to this menu page.
  7. Make image links for all images. Clicking on one should open a new window with a large version of the item clicked.
  8. Complete the learning path on itslearning (videos, links, and exercises).
